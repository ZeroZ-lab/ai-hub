## ❓ 常见问题

### Q1: 我没有 API Key 怎么办？
A: 访问 [Anthropic Console](https://console.anthropic.com/) 注册账号并创建 API Key。新用户通常有免费额度。

### Q2: Python 版本有要求吗？
A: 需要 Python 3.10 或更高版本。运行 `python --version` 检查你的版本。

### Q3: 遇到问题怎么办？
A:
1. 查看 [03_最佳实践.md](docs/03_最佳实践.md) 中的常见问题
2. 检查项目中的 `solution/` 参考实现
3. 在社区讨论区提问
