## 实现步骤

### 准备工作

#### 1. 环境配置
```bash
# 进入项目目录
cd projects/project_XX_name

# 使用 uv 安装依赖
uv sync

# 或者从零开始初始化
uv init
uv add anthropic python-dotenv

# 配置环境变量
cp .env.example .env
# 编辑 .env，填入你的 ANTHROPIC_API_KEY
```

#### 2. 项目结构
```
project_XX_name/
├── README.md           # 本文件
├── pyproject.toml      # uv 项目配置和依赖
├── uv.lock             # 锁定的依赖版本
├── .env.example        # 环境变量示例
├── .env                # 你的环境变量（不要提交）
├── src/
│   ├── __init__.py
│   ├── main.py        # 主程序入口
│   ├── agent.py       # Agent 核心逻辑（待实现）
│   └── utils.py       # 工具函数（可选）
├── tests/
│   ├── __init__.py
│   └── test_agent.py  # 单元测试
├── data/              # 测试数据（如需要）
│   └── sample.txt
└── solution/          # 参考答案（先不要看！）
    └── main.py
```
